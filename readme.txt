Справка по проекту.

1. find_errors_in_sentences.py 
а) принимает на вход путь до файла и 
б) фразу/предложение/слово, которое надо исправить.
в) считает расстояние Дамерау-Левенштейна(основной алгоритм).
г) сохраняет в словарь слова с расстоянием Дамерау-Левенштейна 0, 1, 2.
д) выводит слова и варианты исправлений к нему.
е) надо выбрать нужный вариант исправления и написать его номер.
ё) Если написать - help, - h, то будет справка.

3. frequency_dictionary.py
a) парсит текст, который получил по пути "C:\python_project\errors\new_baza.txt"(А.С.Пушкин "Капитанская дочка").
б) считает количество повторений каждого слова в тексте.

г) выводит слово и количество повторений в тексте в консоль.

г) выводит слова с нужным расстоянием Левенштейна-Дамерау из словаря.
е) надо выбрать нужный вариант исправления и написать его номер.
